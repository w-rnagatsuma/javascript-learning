## 構築手順
- create
```bash
$ npm create vite@latest .

done
```
- Select a framework: `Vanilla`（素のJS）

## 技術スタック

### 言語
- **JavaScript** (ES Modules)
  - モダンなJavaScript構文を使用

### マークアップ・スタイリング
- **HTML5**
- **CSS3**

### 開発ツール
- **Vite** (v7.2.4)
  - 開発サーバー
  - ビルド
  - ホットモジュールリプレイスメント (HMR)

### パッケージマネージャー
- **npm**

## 開発コマンド

```bash
# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```
